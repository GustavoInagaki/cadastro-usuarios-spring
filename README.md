# Cadastro de Usuários - Spring Boot

API REST para cadastro de usuários desenvolvida com Spring Boot.

## Tecnologias
- Java 21
- Spring Boot
- Spring Data JPA
- H2 / PostgreSQL
- Lombok

## Funcionalidades
- Criar usuário
- Listar usuários
- Buscar usuário por ID
- Deletar usuário

## Endpoints
- POST /usuarios
- GET /usuarios
- GET /usuarios/{id}
- DELETE /usuarios/{id}
